# ``SwiftOnig``

A modern Swift wrapper for the Oniguruma regular expression library.

## Overview

SwiftOnig provides a type-safe, concurrency-ready, and idiomatic Swift interface to the powerful Oniguruma regex engine. It supports a wide range of encodings and advanced regex features.
